Glossary¶

ACCC

The Australian Competition & Consumer Commission

ACCC Rules Framework

Refers to the ACCC Rules Framework

CDR

The Consumer Data Right

CDR Bill

A reference to the Treasury Laws Amendment (Consumer Data Right) 2019 Bill. A proposed piece of legislation to enable the delivery of the CDR.

CDR Registry

The Production Registry operated by the ACCC as the government endorsed platform for the efficient communication of Data Recipient accreditation and Data Holder membership information. Registry Runtime (cds-discovery) is intended to be a simulation for Development purposes of this platform.

CDS

See Consumer Data Standards

CDS Engineering

Shortened form of the Consumer Data Standards Engineering Working Group.

CLI

Command Line Interface. A tool intended for execution from a terminal or shell.

client library

Software libraries designed for inclusion within a Data Recipient implementations

Consumer Data Right

The overall initiative that has resulted in these deliverables. Operated by the Consumer Data Standards Australia team within Data61

Consumer Data Standards

The underlying written artefacts which inform how the CDR Ecosystem will operate and how interactions with participants will occur

Consumer Data Standards Australia

The interim standards body, operating within Data61 developing technical standards to deliver on the Consumer Data Right. Official Consumer Data Standards Website

Customer

A customer of a Data Holder, a member of the general public particularly one within Australia

Data Holder

A business which holds data designated by the ACCC as within scope of the Consumer Data Right

Data Recipient

A recipient of data from a Data Holder to deliver additional services on behalf of the Customer

Data61

CSIRO Data61

Engineering Working Group

A reference to the Engineering working group, one of 4 working groups within the project team

project team

See Consumer Data Standards Australia

reference implementation

In the software development process, a reference implementation (or, less frequently, sample implementation or model implementation) is the standard from which all other implementations and corresponding customizations are derived.

server stub

A software “starter kits” designed as a starting point for Data Holder’s to integrate with their existing systems

Standards API

The Standards API

Standards Working Group

A reference to the Standards working group, one of 4 working groups within the project team
