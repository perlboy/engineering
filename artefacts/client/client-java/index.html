CDS Java Client Library (cds-client-java)¶

Summary

The Java Client Library is intended for integration into Data Recipient’s software deployments as part of delivering their product or service. It is intended as a Reference Implementation of the Consumer Data Standards from a Data Recipient (“client”) perspective.

Release Status

Beta

Generation Status

Automatic from CDS Code Generator (cds-codegen)
