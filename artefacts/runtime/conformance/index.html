CDS Conformance Suite (cds-conformance)¶

Note

Development of this documentation is in progress.

Summary

The CDS Conformance Suite is a wrapper around the CDS Java Client Library (cds-client-java) intended to provide a computational validation of inbound and outbound payload data according to the Consumer Data Standards.
