CDS Product Summariser
================================================


+---------------------+---------------------------------------------------------------------------------------+
| Summary             | The Product Summariser is a basic example of the :ref:`cds-client-java` library and   |
|                     | is capable of producing both HTML and PDF based reports utilising a Sphinx based      |
|                     | Docker container. It is intended as a quick example of implementation while serving   |
|                     | the useful purpose of being able to produce printable docs of Product APIs            |
+---------------------+---------------------------------------------------------------------------------------+
| Repositories        | `GitHub`_                                                                             |
|                     | `Docker`_                                                                             |
+---------------------+---------------------------------------------------------------------------------------+
| Release Status      | Pre Release compliant to Standards Product API Specification version 0.9.3            |
+---------------------+---------------------------------------------------------------------------------------+
| Generation Status   | Hand Coded (in <100 lines) and integrated with :ref:`cds-client-java`                 |
+---------------------+---------------------------------------------------------------------------------------+
| Verified Snapshot   | 0.9.3-SNAPSHOT                                                                        |
| Version             |                                                                                       |
+---------------------+---------------------------------------------------------------------------------------+
| Technologies Used   | * Java                                                                                |
|                     | * Maven                                                                               |
+---------------------+---------------------------------------------------------------------------------------+
| Related Artefacts   | * :ref:`cds-client-java`                                                              |
+---------------------+---------------------------------------------------------------------------------------+
